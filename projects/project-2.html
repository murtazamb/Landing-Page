Light-Activated Arduino Car

Designed and built an Arduino-based car prototype that responds to sensor input and light conditions. The system uses custom Arduino code to control motors and LEDs through external circuitry.

Overview

  • Goal: Build a small autonomous car controlled by sensor input using Arduino.
  • Constraints: Limited power, compact size, simple circuitry, and basic components.
  • Your role: Designed the circuit, wrote the Arduino code, assembled and tested the system.
  • Approach: Circuit design → Arduino programming → bench testing → full prototype integration.

What I Learned

  • Using transistors to safely drive motors from an Arduino.
  • Debugging embedded systems through iterative testing.

Tools & Skills

Arduino
Embedded C
Circuit Design
Prototyping
Testing

Results

  • Successfully controlled motors using a TIP120 transistor circuit.
  • Achieved reliable sensor-based activation and LED status indication.
